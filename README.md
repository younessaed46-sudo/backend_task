# Backend Task

Simple backend task using PHP and MySQL.

## Features
- Database connection using PDO
- Fetch users from MySQL database
- Simple backend structure

## Requirements
- XAMPP
- PHP 7+
- MySQL / MariaDB

## How to Run
1. Start Apache and MySQL from XAMPP
2. Import the database
3. Place project in `htdocs`
4. Open in browser:
   http://localhost/backend_task/users.php
